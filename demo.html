jQuery Quiz Plugin

Welcome to the jQuery Quiz Plugin demo! This is a very, very easy to use plugin and requires little knowledge of javascript to apply. In fact, all javascript you'll need in most of the cases is only a one-line code that you can simply copy-and-paste:

$.quiz();

Prerequisites

Before start using, though, you need to set up our few dependencies: jQuery and Mustache.js.

Markup

There are 3 built-in quiz classes (called handler classes) with jQuery Quiz: quiz-single, for single-choice quizzes; quiz-multiple, for multiple-choice quizzes; and quiz-objective, which requires the quiz-taker to write the correct answer. Let's make our first quiz, then!


<div class="quiz-single">
    <p>How much is 2+2?</p>  
    <li class="quiz-option">1</li>
    <li class="quiz-option">2</li>
    <li class="quiz-option">3</li>
    <li class="quiz-answer">4</li>
</div>

The outer div with quiz-single as class says that we have a single-choice quiz, that is, a quiz where there is only one correct answer (and, therefore, the quiz taker can only select one option). The li tags with the class quiz-answer correspond to the correct answers of the quiz, and the others are the wrong answers. Only elements with the class quiz-option will be considered options, but don't worry: elements with quiz-answer class will automatically have quiz-option class added (so you can style then easily!).

    • Ok, the quiz is there with the options, but where can I submit it? Yeah, you forgot to add the controls. There are 5 predefined classes of controls:

  • quiz-submit: when clicked, submits you answer.
  • quiz-clear: when clicked, clears the answers checked.
  • quiz-show-answer: when clicked, show the correct answer.
  • quiz-toggle-hint: when clicked, shows/hides a hint (any element with class quiz-hint) for solving the problem.
  • quiz-explanation: a element that is only showed after the quiz taker submits the correct answer, explaining why are the options submitted correct and the others wrong.

    • All of them are of optional usage, although I don't thing you can make anything useful without a quiz-submit :P. Let's then try them all at the same time, so you can see what do they do!

    JavaScript API

    Well, as said in the first paragraph, all JavaScript you'll need in most of the cases is simply

    $.quiz();

    which looks for all elements with a handler class (by default, quiz-single, quiz-multiple, and quiz-objective) and transform them into quizzes. However, there are times when you need to be more specific on which quizzes to transform or not. To do this, you can simply use standard jQuery way:

    $('#quiz-id').quiz();

    or also

    $.quiz($('#quiz-id'));

    You can also specify, instead of a element, a context in which to search, like 

    $('#quizzes').quiz(); // search for quizzes only inside #quizzes.
    $.quiz($('#quizzes')); // same as above.

    Extending our plugin

    Besides our three predefined handlers, you can also create your own, specific handler to fully meet your needs. To do this, simply add it to the $.quiz.handlers object, with a key containing the css classname for the handler (e.g. 'quiz-something'), and the value, the name of your JavaScript type.

    The API will pass to the constructor of your class two parameters: question, a jQuery object pointing to the question to transform, and idQuestion, which is its id, or an distinct integer if it doesn't have and id. The objects of the class must have a method init(), which will render the quiz properly.
